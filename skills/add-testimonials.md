# Skill: Add Testimonials

## Purpose
Add or update testimonials on any UC35 client site.

## Trigger
"add testimonials to [client name]" or
"update testimonials for [client name]"

## Steps
1. Open target HTML file
2. Locate existing testimonials section
3. If adding: insert new testimonial card using
   existing card HTML as template
4. If updating: replace specified testimonial content
5. Testimonial card must include:
   - Quote text (2-3 sentences max)
   - Customer first name + last initial
   - City and state
   - Star rating (5 stars default, use CSS or emoji)
6. Save file and confirm change
