# Skill: New Client Site

## Purpose
Full end-to-end workflow for building a new client site
from intake to delivery-ready file.

## Trigger
"new client site" or "start [client name] project"

## Steps
1. Confirm you have all required intake info:
   - Business name, niche, city, state
   - Accent color preference
   - Services (3 minimum)
   - Contact info
   - Any special notes

2. If missing info — ask for it before proceeding

3. Activate copy.md — generate all copy first

4. Activate builder.md — build full HTML site using
   approved copy and intake info

5. Save to ~/Documents/UC35/clients/[client-name]/index.html

6. Activate review.md — run full quality audit

7. Fix any issues found in review

8. Confirm: "Site ready for Figma comparison and
   client delivery"
