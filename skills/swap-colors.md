# Skill: Swap Colors

## Purpose
Rebrand any existing site template to a new client's
accent color in one pass.

## Trigger
"swap colors for [client name] to [color/hex]"

## Steps
1. Open the target HTML file
2. Identify current accent color hex value
3. Find and replace all instances with new color
4. Check: buttons, borders, highlights, hover states,
   stat boxes, section accents
5. Confirm replacement count
6. Save file
7. Output: "Color swap complete — [old hex] → [new hex],
   [X] instances replaced"
