# Skill: Deploy Checklist

## Purpose
Final check before sending preview link to client.

## Trigger
"run deploy checklist" or automatically at end of
every build

## Checklist
- [ ] File saved to correct client folder
- [ ] File opens in browser without errors
- [ ] All sections visible on desktop
- [ ] All sections visible on mobile (375px)
- [ ] No broken links in navigation
- [ ] Contact form fields all present
- [ ] Accent color consistent throughout
- [ ] No lorem ipsum
- [ ] No placeholder URLs
- [ ] GSAP animations trigger on scroll
- [ ] Footer copyright year is current

## Output
"Deploy checklist passed — ready to send to client."
or list of items to fix.
