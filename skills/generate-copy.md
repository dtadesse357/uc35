# Skill: Generate Copy

## Purpose
Quickly generate all website copy for any construction
business without building the full site.

## Trigger
"generate copy for [business name]"

## Steps
1. Activate copy.md agent
2. Collect required info (name, niche, city, services)
3. Output all copy sections in order:
   - Nav links
   - Hero headline + subheadline
   - 3 service card titles + descriptions
   - About paragraph + 3 stats
   - 2 testimonials
   - Contact headline
   - Footer tagline
4. Format output clearly with section labels
5. Ask: "Approve this copy or request changes?"
